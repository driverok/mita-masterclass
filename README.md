# MITA - Mini Issue Tracker Application

Demo project for Claude Code Masterclass showcasing GitHub Actions + Vercel CI/CD integration.

## Features

- Issue tracking with priorities and statuses
- Sprint management with burndown charts
- Real-time updates
- Mobile responsive design

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Database:** Prisma + SQLite
- **Styling:** Tailwind CSS
- **CI/CD:** GitHub Actions + Vercel

## Getting Started

```bash
npm install
npx prisma generate
npm run dev
```

## CI/CD Setup

This project uses:
- **Claude Code** for AI-assisted issue resolution
- **Vercel** for preview and production deployments
